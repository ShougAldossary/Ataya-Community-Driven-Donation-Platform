♻️ Ataya — Community Driven Donation Platform

A desktop application that connects people to donate, exchange, and discover second-hand items within their community. 
Built with Java Swing and MySQL, it features a full user-facing interface alongside an admin dashboard for moderation and reporting.

📖 About the Project

Ataya is a community-driven platform where users can list pre-owned items (clothing, bags, supplies, and more) for others to discover and request. 
An admin layer ensures quality control through item approval, user management, and reporting tools.
This project was developed as a Software Engineering course project, applying real-world software design principles including layered architecture, session management, input validation, and database-driven UI.
